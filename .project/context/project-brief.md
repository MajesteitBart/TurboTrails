# Project Brief

`Turbo Trails 2.0` is a browser-based 2D side-scrolling stunt/trialbike game for kids. Players ride bikes and stunt vehicles through short obstacle levels, collect coins and chests, perform flips and balance stunts, hit checkpoints, finish quickly, earn stars, and unlock vehicles.

The game must be original. It may use trialbike games as genre reference, but it must not copy names, assets, exact UI, exact levels, logos, sounds, or layouts from Stuntbike Extreme or any other commercial game.

MVP target: 3 worlds, 12 levels, 4 vehicles, keyboard controls, touch controls, checkpoints, finish flow, results, garage, local saves, and tests.
